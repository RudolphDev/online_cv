DigCNV

GitHub Repository

2021 - Présent R / BioConductor RudolphDev

R Package BioConductor

DigCNV est un package R conçu pour le contrôle qualité des CNVs (Copy Number Variations). Il permet d'éliminer les CNVs artefactuels identifiés par les callers PennCNV et QuantiSNP, améliorant ainsi la précision des analyses génomiques. DigCNV is an R package designed for quality control of CNVs (Copy Number Variations). It eliminates artifactual CNVs identified by PennCNV and QuantiSNP callers, thereby improving the accuracy of genomic analyses.

Fonctionnalités principales : Main Features:

  • Détection des CNVs artefactuels
  • Filtrage basé sur des critères statistiques
  • Visualisation des résultats
  • Intégration avec les pipelines existants
  • Documentation complète et exemples
  • Detection of artifactual CNVs
  • Filtering based on statistical criteria
  • Result visualization
  • Integration with existing pipelines
  • Comprehensive documentation and examples
